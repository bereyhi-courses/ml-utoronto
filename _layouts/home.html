Course Syllabus

The course consists of three main parts, namely Learning from Data, Classification and Regression, and Introduction to Advanced Topics in ML. The contents taught in each part are listed below:

Part I: Learning from Data

  1. Concept of Learning and Different Problems in ML
  2. Clustering
    • Problem of Clustering
    • K-means Clustering Algorithm
    • Notion of Distortion
    • Soft K-means Clustering
  3. Learning Data Distribution
    • Statistical View on Data
    • Modeling Data Distribution via Parameterized Model
    • Maximum Likelihood Learning
    • Gaussian Maximum Likelihood
    • Gaussian Mixture Model
    • Data Generation
  4. Dimensionality Reduction and Data Representation
    • Latent Representation for High Dimensional Data
    • Principle Component Analysis (PCA)
    • Data Compression and Filtering via PCA
    • Introduction to Nonlinear PCA

Part II: Classification and Regression

  1. Linear Regression and Classification
    • Notion of Loss in Supervised Settings
    • Linear Regression: Optimal Model
    • Logistic Regression: Optimal Model
    • Gradient Descent Algorithm
  2. Support Vector Machine
    • Classification with Maximum Margin
    • Support Vector Classifier (SVC)
    • Dimensionality Expansion via Embedding Function
    • Support Vector Machine (SVM)
    • Various Kernels for SVM
    • SVM with Soft Margin
  3. Neural Networks
    • Neural Networks (NNs) as Parameterized Models
    • Necessity of Nonlinearity in (NNs)
    • Expressing Binary Functions via NNs
    • Approximating Analytic Functions via NNs
    • Universal Approximation Theorem
  4. Generalization
    • Formulating Generalization Error
    • Overfitting and Underfitting
    • Bias-Variance Trade-off and Bayes Error Bound
    • Regularization and Ensemble Learning
  5. Convolution
    • Convolution as a Linear Operator
    • Designing a Parameterized Model via Convolution
    • Sample Convolutional NN

Part III: Advance Topics in ML

  1. Introduction to Sequence Processing
    • Processing Sequences with Temporal Correlation
    • Quick Introduction to RNNs and Transformers
  2. Introduction to Generative Models
    • Learning How to Generate Data
    • Implicit and Explicit Generative Models
  3. Introduction to Reinforcement Learning
    • Components and Objectives in Reinforcement Learning (RL)
    • Tabular RL versus RL with Function Approximation
